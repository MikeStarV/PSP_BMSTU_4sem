# PSP_BMSTU_4sem
Планирование технического обслуживания авиапарка.
Студент: Стариков М.В Группа: ИУ5-43Б Преподаватель: Аксенова М.В.

Данный репозиторий содержит цикл лабораторных работ по разработке веб-приложения «Планирование технического обслуживания авиапарка».

Структура проекта
Каждая лабораторная работа находится в отдельной ветке:

Ветка	Лабораторная работа	Описание
lab1	№1	Интерфейс (HTML/CSS)
lab2	№2	Логика (JavaScript)
lab3	№3	Архитектура SPA
lab4	№4	REST API (Express)
lab5	№5	Интеграция (XHR, CORS)
lab6	№6	Модернизация (fetch, Vite)
